## Описание

Упражнение по express.js

## Локальный запуск

1. Склонировать репозиторий
2. Доставить отсутствющие модули npm

    ```bash
        npm install
    ```

3. Запустить локальный сервер

    ```bash
        npm run start
    ```

  для разработки возможен запуск с develop'оп окружением

  ```bash
      npm run dev
  ```
